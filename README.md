kernel driver simple para leer y escribir en la memoria, se puede utilizar para muchas cosas, recorda instalar wdk y usar el bcdedit -set testsigning off y bcdedit -debug on

FEATURES: 
- memory read/write fron ring 0 (kernel) < sistema de memoria fisica
- set privilege (usermode)
- pattern scanning (aob)
- string cleaner (aob / usermode)
